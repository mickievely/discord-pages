MyDiscordApp — 연결된 역할 확인 페이지

이 페이지는 Discord 애플리케이션의 Linked Role Verification URL 용으로 사용됩니다.

사용 방법

  1. Discord 개발자 포털 → 애플리케이션 선택
  2. OAuth2 → Linked Roles 메뉴에서 Verification URL 입력
  3. 이 페이지의 주소를 사용: https://BGSIAutoFarm.github.io/discord-pages/index.html
  4. Discord가 보낸 ?token=... 파라미터는 서버에서 검증해야 합니다.

테스트

URL에 ?token=123abc 처럼 붙여 확인할 수 있습니다.
