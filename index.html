Volt 설치 및 사용

영상 가이드를 따라 하면 한국어 안내로 빠르게 설정할 수 있습니다.

이렇게 하시면 됩니다

  1. 아래 유튜브 안내 영상을 열어 순서대로 따라 해 주세요.
  2. 한국어 지원이 되므로, 화면에 나오는 그대로 진행하시면 됩니다.

영상 링크는 공식 채널 또는 구매처에서 안내받은 주소를 사용해 주세요.

키훅 (kiciahook) 사용법

디스코드에서 키를 등록하고, 발급된 스크립트를 복사해 사용합니다.

순서

  1. kiciahook 디스코드 서버에 접속합니다.
  2. 서버 안에서 script 관련 채널로 이동합니다.
  3. Redeem Key에 본인이 구매한 키를 입력합니다.
  4. Get Script를 눌러 키가 포함된 스크립트를 복사한 뒤 사용합니다.

Wave 설치 및 사용

사전에 필요한 프로그램을 준비한 뒤, 설치 파일을 실행하고 라이선스를 등록합니다.

사전 준비

  • 안티바이러스 — 설치·실행이 막히면 일시적으로 끄거나, 기존 방화벽 앱에서 실시간 보호만 꺼 보세요. (환경에 따라 생략 가능)
  • Windows Defender 제어 도구 (참고) — Defender Control v1.5
  • .NET SDK 10 — Windows x64 설치 프로그램
  • Visual C++ 재배포 패키지 — VC++ x64
  • Node.js — 공식 프리빌트 설치 안내

설치

  1. 라이선스를 구매한 뒤 클라이언트를 받습니다 — getwave.gg
  2. Wave-Setup.exe를 실행합니다. (Bloxstrap이 함께 설치될 수 있습니다.)
  3. Wave를 실행하고 회원가입을 완료합니다.
  4. 화면의 키 아이콘을 눌러 라이선스를 입력합니다.

실행 시 참고

  • Wave.exe를 실행한 뒤 안내에 따라 삽입합니다.
인젝션(삽입)은 Roblox 로비에서 진행해 주세요. 게임 안이 아닌 로비 상태에서 진행하는 것이 안전합니다.
Wave로 스크립트 실행 준비가 끝나면, 이후 단계는 Wave 안내에 맞춰 진행하면 됩니다.

Potassium 사용법

공식 디스코드의 봇 명령으로만 받은 파일을 사용하세요.

순서

  1. 공식 디스코드에 들어갑니다 — Potassium 디스코드
  2. 서버에서 psm 봇을 찾습니다.
  3. /register 명령으로 등록을 마칩니다.
  4. 등록이 끝나면 /download로 공식 클라이언트를 받습니다.
다운로드는 반드시 공식 디스코드에서만 진행하세요. 다른 경로의 파일은 사용하지 마세요.
